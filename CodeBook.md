# CodeBook
This CodeBook describes each variable you can find in result.txt.

## Activities and Subjects

Subject and Activity variable identify unique couples. 

Variables | Explanation | Type | Values |
--- | --- | --- | --- |
 **Subject** | The subject ID | Integer | |
 **Activity** | The activity name | String | Walking, Walking Upstairs, Walking Downstairs, Sitting, Standing, Laying
